## pager

~~~
1페이지 1-10
2페이지 11-20

* 시작번호 = (현재페이지 -1) * 페이지당 게시물 수 +1
  ex) 1페이지 = (1 -1 ) * 10 +1 = 1
      2페이지 = (2 -1 ) * 10 +1 = 11
      3페이지 = (3 -1 ) * 10 +1 = 21

* 전체 페이지 블록 수 = 전제페이지 / 10
  ex) 91/10 = 9.1 ==> 10페이지
  
* 현재 페이지가 속한 블록 = (현재페이지 -1) / 페이지 블록 단위 + 1
  ex) 1페이지 ==> 몇번째 블록?
     (1 -1) / 10 + 1 => 1
      9페이지
     (9 -1) /10 + 1 => 1
      11페이지
     (11 -1) /10 + 1 => 2페이지

* 페이지 블록의 시작번호 =  (현재블록 -1) * 블록 단위 +1
  ex) 1블록 ==> (1-1) * 10 + 1 => 1
      2블록 ==> (2-1) * 10 + 1 => 11
      
* 페이지 블록의 끝번호  =  (블록시작번호 + 블록단위 -1 )
  ex ) 1블록 ==> (1+10-1) => 10
       2블록 ==> (2+10-1) => 21-1 => 20
       
~~~
