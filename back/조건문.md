### 조건문( for문 )

~~~
for문 (초기화; 조건식; 증감식) {
  // 조건식이 true 일때 수행될 코드
}

* 여기서 조건식은 생략이 가능하다.
-> 조건식이 생략되면 true로 간주된다.

for( ; ; ) { /*수행할 코드*/ }  --> true 로 무한반복
for(int i=0; ; ) { /*수행할 코드*/ } --> 변수 i는 for문내에서만 유효
for(int i=1, j=1 ; i<10 || i*j(..), j+=2 ) { /*수행할 코드*/ } --> ,를 이용하여 하나 이상의 변수를 선언 가능 
~~~
